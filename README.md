# Sapi3D

3D building visualization with interactive exploration, user authentication, and achievement system.

## Quick Start

**Mode:** `dev` or `prod`

```bash
./run_[mode].sh          # Start services
./stop_[mode].sh         # Stop services
./logs_[mode].sh         # View logs
```

**Access:**
- Frontend: http://localhost:3000
- API Docs: http://localhost:8000/docs
- Database: localhost:5432
